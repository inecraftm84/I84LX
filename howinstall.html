x64 Install Guide

1. Check if CLI Installer is installed

Run the following command:

dpkg -l | grep cli-installer

If you see cli-installer-antix in the output, you can skip the next step.

If it is not installed, run the following command:

sudo apt install cli-installer-antix

2. Run the installer

Execute the command:

sudo cli-installer

The installer will ask: "Do you want to repartition the disk?" â†’ type Y

Select the partitioning scheme (e.g., GPT).

Click "New" for free space, set the partition size (if this is the only OS, use the full disk size), then click "Write".

3. Configure partitions

Choose the root partition (e.g., sda1).

Select the file system (ext4 is recommended).

Optional: You can choose to use a separate /home partition according to your preference.

If asked "Preserve data in /home?", usually choose NO.

4. Continue installation

The installer will erase data on the selected partition. Confirm to continue.

If you're using antiX-net, note that older x64 versions are based on antiX-base, while newer antiX-core x32 versions are based on antiX-core.

5. Optional packages

The installer may ask if you'd like to install additional packages via cli-aptiX. Choose according to your needs.

Confirm to continue and complete the installation. Please wait.

6. Where should GRUB be installed?

1. MBR (32-bit recommended)

2. Root partition (your choice)

3. ESP (EFI System Partition, 64-bit recommended)

4. Do not install GRUB (for advanced users)

7. Computer Name?

Choose a name for your computer, such as I84, computer, antix1, mycomputer, XxGamerXx.

8. System locale

Your choice, e.g., Taiwan: zh_TW, China: zh_CN, US: en_US.

9. Set up keyboard

If your keyboard layout is not QWERTY, you can select your preferred layout.

10. Console layout

If you dislike the default console font, you can choose a different one.

11. Timezone

It is recommended to set the correct timezone.

12. Is this a snapshot?

If asked, select "Yes".
