x64 Install Guide

1. Check if CLI Installer is installed

Run the command:

dpkg -l | grep cli-installer

If you see cli-installer-antix in the output, you can skip the next step.

If not installed, run:

sudo apt install cli-installer-antix

2. Run the installer

Run:

sudo cli-installer

The installer will ask: "Do you want to repartition the disk?" â†’ type Y

Select the label type (e.g., GPT).

Click "New" for free space, set partition size (if no other OS, use full disk size), then click "Write".

3. Configure partitions

Choose root partition (e.g., sda1).

Select the file system (ext4 recommended).

Optional: use a separate /home partition? Choose according to your preference.

If asked "Preserve data in /home?", usually NO.

4. Continue installation

The installer will destroy data on the selected partition. Confirm to continue.

If running antiX-net, note: x64 versions are built on antiX-base,x32 versions are built on antiX-core

5. Optional packages

The installer may ask if you want to install packages via cli-aptiX. Choose according to your needs.

Confirm to continue and complete the installation.WAIT.
